# Workflow

- **Task Management:** All tasks are managed via the external Obsidian Kanban board (`/data/Documents/i_obsidian_vault/hevy-companion/Task-Board.md`).
- **Prioritization:** Always check and work on tasks in the **Active** column first. Only pull from the **Backlog** if the Active column is empty.
- **Development Flow:** 
  1. Analyze features across Database, Backend (Kotlin), and Security/Config layers.
  2. Implement features idiomatically and securely.
  3. Validate against project constraints (e.g., specific ports, null-safety).
